# Migration helpers (temporary)

This folder holds **snapshots and playbooks** for one-off repo changes (for example aligning a feature branch with `main` without carrying a duplicate Vite app).

- **Component 3 frontend snapshot:** see [`component3-frontend/README.md`](component3-frontend/README.md).

After the merge is complete and files live under `frontend/` again, you can delete the `migration/component3-frontend/` tree in a follow-up commit if you no longer need the snapshot.
